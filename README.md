# University Examination Management System – UML Diagrams

This project contains UML diagrams for a University Examination Management System.

## Diagrams Included

1. Use Case Diagram
2. Class Diagram
3. Sequence Diagram
4. Collaboration Diagram
5. Activity Diagram
6. Statechart Diagram
7. Component Diagram
8. Deployment Diagram

## System Description

The University Examination System manages the entire exam workflow including:

- Exam timetable creation
- Hall allocation
- Invigilator assignment
- Examination conduction
- Evaluation of answer scripts
- Result processing
- Result publication
- Revaluation process

## Actors in the System

- Administrator / Exam Coordinator
- Faculty
- Controller of Examinations
- Invigilator
- Student

## Tools Used

- Draw.io / Lucidchart (for UML diagrams)
- GitHub (for version control)
